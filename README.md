# URL-Shortener

Welcome to the URL Shortener App! This application provides a simple way to shorten long URLs, making them easier to share and manage. Additionally, it tracks the total number of clicks on each shortened URL and logs the timestamps of each click.

# Features

**URL Shortener:** Convert long URLs into short,copy them, easy-to-share links.

**URL Analytic:** Monitor the number of times each shortened URL has been clicked by typing the shortened URL ID and record the exact time each shortened URL is accessed.

# Technologies Used

Backend: Node.js, Express.js

Database: MongoDB

Frontend: React.Js

Other: dotenv (for environment variable management)
